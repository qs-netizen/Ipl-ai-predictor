# 🏏 IPL AI Match Predictor

An advanced Machine Learning project that predicts IPL match winners using Python, XGBoost, Streamlit, and data analytics.

---

# 🚀 Features

* IPL Match Winner Prediction
* Machine Learning Model (XGBoost)
* Team Statistics Dashboard
* Winning Probability Graphs
* Head-to-Head Analysis
* Interactive Streamlit UI
* IPL Match Data Visualization
* CSV Dataset Support
* Professional Dark Theme UI

---

# 🧠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* Streamlit
* Plotly
* Joblib

# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/ipl-ai-predictor.git
```

## Move into Project Folder

```bash
cd ipl-ai-predictor
```

## Install Requirements

```bash
pip install -r requirements.txt
```

---

# 📊 Dataset

Place datasets inside the `data/` folder.

Required files:

* `matches.csv`
* `deliveries.csv`

Dataset Source:

* IPL Kaggle Dataset
* Cricsheet IPL Data

---

# 🤖 Train the ML Model

Run:

```bash
python train_model.py
```

After training, these files will be created automatically:

```text
models/
├── ipl_model.pkl
├── encoders.pkl
```

---

# 🌐 Run the Streamlit App

```bash
streamlit run app.py
```

Then open:

```text
http://localhost:8501
```
